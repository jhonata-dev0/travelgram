📸 Travelgram
O Travelgram é uma página de perfil de rede social focada em viagens, desenvolvida como um desafio da Rocketseat. O projeto apresenta um layout moderno, totalmente responsivo, onde usuários podem visualizar uma galeria de fotos de destinos internacionais de forma organizada e elegante.

🚀 Tecnologias
Este projeto foi construído utilizando as seguintes tecnologias:

HTML5: Estrutura semântica para melhor acessibilidade e SEO.

CSS3: Uso de variáveis (:root), Flexbox e CSS Grid.

CSS Modular: Organização de estilos dividida por componentes para facilitar a manutenção.

Design Responsivo: Adaptado para diferentes resoluções (Mobile, Tablet e Desktop).

Google Fonts: Utilização das fontes Poppins e Roboto.

🛠️ Funcionalidades
Galeria Dinâmica: Grid de imagens que se ajusta automaticamente ao tamanho da tela usando auto-fit.

Header Adaptável: Menu de navegação que se transforma em layout de coluna em dispositivos menores.

Interface Limpa: Foco na experiência do usuário e na legibilidade.

🎨 Design
O layout foi projetado para transmitir uma sensação de organização e inspiração.

Cores principais: Tons de cinza suave para o fundo e cores vibrantes para interação.

Tipografia: Hierarquia clara entre títulos e textos de apoio.
